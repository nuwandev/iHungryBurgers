# Screenshots Folder

This folder is for storing application screenshots for the README.

## Required Screenshots

Please add the following screenshots to make the README complete:

1. **home-screen.png** - Main navigation screen with all menu options
2. **place-order.png** - Order placement form showing customer details and quantity input
3. **search-customer.png** - Customer search interface
4. **view-orders.png** - List of orders with different status filters
5. **update-order.png** - Order status update interface
6. **best-customer.png** - Best customer analytics/identification screen

## Screenshot Guidelines

- **Format:** PNG preferred (supports transparency)
- **Resolution:** 1920x1080 or similar high-quality resolution
- **Content:** Make sure to show the full application window
- **Privacy:** Remove any sensitive test data if present
- **Naming:** Use lowercase with hyphens (as listed above)

## How to Take Screenshots

1. Run the application
2. Navigate to each screen
3. Take a full window screenshot (or use a screenshot tool)
4. Save the images in this folder with the names above
5. The README will automatically display them
